### Front end using "npm run dev", backend using "npx nodemon"
- Can open mutiple nstance of port 5173 to have a group chat
- by default all sessions have public rooms which cant be removed, they are like announcements
- Then can add, custom rooms, with inbuilt vibrant colours in room name and messages as well
- You can be on a different room, but if you have joined a custom room, your messages are safe and will be restored once you came back.
- Nice alignment of messages ent by you vs the one sent by others
- Have a feature to clear messages from your end
- Also get to the newest msgs just by clicking on icon
